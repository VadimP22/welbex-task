# Что нужно?

docker, npm

# Как собрать?

```
git clone https://github.com/VadimP22/welbex-task.git
```

```
cd welbex-task
```

```
npm run dependencies
```

```
npm run build
```

```
npm run dockerize
```

# Как запустить?

```
docker compose up
```

# Как посмотреть?

http://127.0.0.1:8088

HTTPS не работает! Только HTTP!
